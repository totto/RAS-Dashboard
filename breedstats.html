Breed Statistics Snippets

Dependencies

  • jQuery
  • Google Chart

Usage

BREEDSTATS.draw();

Draws breed statistics chart at the script tag location.
      BREEDSTATS.draw( breed, graphtypes );
      BREEDSTATS.draw( breed, graphtypes, from-year, to-year, generations );
breed Name of the breed. E.g. 'Rottweiler'.
graphtypes The types of graphs you want to display. Possible values are
  • Innavl
  • Stamtavlekompletthet
  • Kull
  • HD
Show multiple graphs by writing multiple values in this field. E.g. "Innavl Kull HD"
from-year, to-year Start and end years to be charted in the graph. Default: 2004 and 2014.
generations Number of generations to use for calculations of inbreeding. Default: 6.
Example
        <script>
          BREEDSTATS.draw( 'Rottweiler', 'Innavl', 2010, 2013, 5 );
        </script>

BREEDSTATS.setBaseUrl()

Setting the base url for the graph endpoints.
        <script>
          BREEDSTATS.setBaseUrl( 'yourdomain.com/dogpopulation/graph/' );
        </script>

Baseurl must be set before drawing the graphs.

Note: yourdomain.com server needs to accept requests from the sender domain.
